# 📖 Glossário - Hackers do Bem (1ª Onda)

## 🔹 A
- **ACK** - Confirmação (acknowledgment) no protocolo TCP
- **Active Directory** - Serviço de diretório da Microsoft para gerenciamento de domínios e usuários
- **AES** - Advanced Encryption Standard, algoritmo de criptografia simétrica
- **APT** - Advanced Persistent Threat, ameaça persistente avançada
- **ARP** - Address Resolution Protocol, resolve IP → MAC
- **Assembly** - Linguagem de baixo nível próxima do código de máquina

## 🔹 B
- **Backdoor** - Porta dos fundos, acesso não autorizado escondido
- **Baiting** - Técnica de engenharia social com iscas (ex: pendrive infectado)
- **Black Hat** - Hacker malicioso que age ilegalmente
- **Botnet** - Rede de dispositivos infectados controlados remotamente
- **Broadcast** - Comunicação de um para todos na rede
- **Btrfs** - B-Tree File System, sistema de arquivos Linux com snapshots

## 🔹 C
- **Cabeçalho** - Header, informações de controle adicionadas em cada camada
- **Cache DNS** - Armazenamento temporário de consultas DNS
- **Cavalo de Troia** - Trojan, malware disfarçado de programa legítimo
- **Certificado Digital** - Documento eletrônico que associa uma chave pública a uma entidade
- **Chave Assimétrica** - Par de chaves (pública e privada) para criptografia
- **Chave Simétrica** - Mesma chave para criptografar e descriptografar
- **CIDR** - Classless Inter-Domain Routing, notação de máscara (/24)
- **Cifra** - Método de criptografar uma mensagem
- **Cifra de César** - Substituição por deslocamento no alfabeto
- **CIS Controls** - Center for Internet Security, controles de segurança prioritários
- **Classful** - Roteamento por classes (A, B, C)
- **Classless** - Roteamento sem classes (CIDR)
- **Cluster** - Conjunto de setores no armazenamento
- **Comando** - Instrução executada no terminal
- **Compilador** - Traduz código inteiro de uma vez para linguagem de máquina
- **Confidencialidade** - Pilar CIA, só quem deve ver acessa
- **Controle de Acesso** - Mecanismo que restringe acesso a recursos
- **Criptoanálise** - Estudo de quebrar criptografia
- **Criptografia** - Técnica de codificar informações
- **CVE** - Common Vulnerabilities and Exposures, identificador de vulnerabilidades
- **CVSS** - Common Vulnerability Scoring System, pontuação de gravidade (0-10)
- **CWE** - Common Weakness Enumeration, categorização de falhas

## 🔹 D
- **Dados** - Informações brutas sem tratamento
- **DDoS** - Distributed Denial of Service, ataque distribuído de negação de serviço
- **Debian** - Distribuição Linux base para Ubuntu, Kali
- **DES** - Data Encryption Standard, algoritmo de criptografia antigo
- **DevSecOps** - Integração de segurança no desenvolvimento (Development, Security, Operations)
- **DHCP** - Dynamic Host Configuration Protocol, atribui IP automático
- **Disponibilidade** - Pilar CIA, sistema funcionando quando necessário
- **DNS** - Domain Name System, converte nomes em IPs
- **DORA** - Discover, Offer, Request, Acknowledge (processo DHCP)
- **DoS** - Denial of Service, negação de serviço

## 🔹 E
- **E-mail** - Correio eletrônico
- **Engenharia Social** - Manipulação psicológica para obter informações
- **Enigma** - Máquina de criptografia alemã na 2ª Guerra
- **Esteganografia** - Ocultar mensagem dentro de outra (ex: em imagem)
- **Ethernet** - Padrão de rede com fio
- **Ethical Hacking** - Hacking ético autorizado
- **Exfiltração** - Roubo de dados para fora da rede
- **Exploit** - Código que explora uma vulnerabilidade
- **Ext4** - Fourth Extended File System, padrão no Linux

## 🔹 F
- **FAT32** - File Allocation Table 32, sistema de arquivos antigo (limite 4GB)
- **Firewall** - Equipamento que filtra tráfego baseado em regras
- **Força Bruta** - Ataque que testa todas as combinações possíveis
- **Framework** - Conjunto de diretrizes e boas práticas
- **FTP** - File Transfer Protocol, transferência de arquivos (porta 21)

## 🔹 G
- **GET** - Método HTTP para solicitar dados
- **Glossário** - Lista de termos e definições
- **Governança** - Estrutura de decisão e direção da segurança
- **GPT** - GUID Partition Table, tabela de partições moderna
- **GRC** - Governança, Risco e Compliance
- **Gray Hat** - Hacker entre o bem e o mal (sem autorização, mas sem intenção maliciosa)
- **Grep** - Comando Linux para buscar texto em arquivos

## 🔹 H
- **Hacker** - Pessoa com habilidades avançadas em tecnologia
- **Hardware** - Parte física do computador
- **Hash** - Função matemática unidirecional (ex: MD5, SHA)
- **Hubs** - Equipamento camada 1, repete sinal para todas as portas
- **HTTP** - HyperText Transfer Protocol (porta 80, texto claro)
- **HTTPS** - HTTP com SSL/TLS (porta 443, seguro)

## 🔹 I
- **IaaS** - Infrastructure as a Service (infraestrutura como serviço)
- **IANA** - Internet Assigned Numbers Authority, coordena IPs globalmente
- **IDS/IPS** - Intrusion Detection/Prevention System
- **IEEE** - Instituto de Engenheiros Eletricistas e Eletrônicos
- **IETF** - Internet Engineering Task Force
- **IMAP** - Internet Message Access Protocol (porta 143, mantém e-mails no servidor)
- **Injeção** - Ataque que insere código malicioso (ex: SQL injection)
- **Integridade** - Pilar CIA, informação não alterada sem permissão
- **Interpretador** - Traduz e executa linha por linha
- **IoT** - Internet of Things, internet das coisas
- **IP** - Internet Protocol, endereço lógico
- **IPv4** - Versão 4 do IP (32 bits)
- **IPv6** - Versão 6 do IP (128 bits)
- **ISO 27001** - Norma internacional para gestão de segurança da informação

## 🔹 J
- **Janelamento** - Controle de fluxo no TCP
- **Journaling** - Registro de alterações no sistema de arquivos

## 🔹 K
- **Kali Linux** - Distro focada em pentest
- **Kernel** - Núcleo do sistema operacional
- **Keylogger** - Malware que captura teclas digitadas

## 🔹 L
- **Largura de banda** - Capacidade máxima de transmissão
- **Latência** - Atraso na transmissão
- **LDAP** - Lightweight Directory Access Protocol, acesso a diretórios
- **LGPD** - Lei Geral de Proteção de Dados (Lei 13.709/18)
- **Link Local** - Endereço IPv6 FE80::/3, comunicação local
- **Linux** - Sistema operacional open source
- **Loopback** - Endereço 127.0.0.1 (IPv4) ou ::1 (IPv6)
- **LS** - Comando Linux para listar arquivos

## 🔹 M
- **MAC Address** - Media Access Control, endereço físico da placa de rede (48 bits)
- **Malware** - Software malicioso (vírus, worm, trojan, ransomware)
- **MAN** - Metropolitan Area Network, rede metropolitana
- **Máscara de rede** - Define rede e host (ex: 255.255.255.0)
- **MBR** - Master Boot Record, setor de inicialização antigo
- **MD5** - Message Digest 5, algoritmo de hash (inseguro atualmente)
- **MITM** - Man-in-the-Middle, ataque de interceptação
- **MITRE ATT&CK** - Framework de táticas e técnicas de ataque
- **Multicast** - Comunicação de um para um grupo

## 🔹 N
- **NAT** - Network Address Translation, tradução IP privado/público
- **NIST** - National Institute of Standards and Technology
- **NIST CSF** - Cybersecurity Framework do NIST (Identificar, Proteger, Detectar, Responder, Recuperar)
- **Nmap** - Ferramenta de scan de redes
- **NoSQL** - Banco não relacional (MongoDB, Cassandra)
- **NTFS** - New Technology File System, sistema de arquivos Windows com permissões
- **NTP** - Network Time Protocol, sincronização de horário

## 🔹 O
- **Octeto** - 8 bits
- **On-premise** - Infraestrutura local (dentro da empresa)
- **OpenVAS** - Open Vulnerability Assessment System, scanner de vulnerabilidades
- **OSI** - Open Systems Interconnection, modelo de 7 camadas
- **OWASP** - Open Web Application Security Project
- **OWASP Top 10** - As 10 principais vulnerabilidades web

## 🔹 P
- **PaaS** - Platform as a Service (plataforma como serviço)
- **Pacote** - PDU da camada de rede
- **PAN** - Personal Area Network, rede pessoal (Bluetooth)
- **Patch** - Correção de software
- **PDU** - Protocol Data Unit, unidade de dados em cada camada
- **Pentest** - Teste de penetração, invasão autorizada
- **Permissão** - Direito de acesso a arquivos/pastas (Linux: rwx)
- **Phishing** - Fraude por mensagens falsas
- **Pilares CIA** - Confidencialidade, Integridade, Disponibilidade
- **Ping** - Teste de conectividade
- **PKI** - Public Key Infrastructure, infraestrutura de chave pública
- **POP3** - Post Office Protocol (porta 110, baixa e apaga do servidor)
- **Porta** - Identificador de serviço na camada de transporte (0-65535)
- **POST** - Método HTTP para enviar dados
- **Pretexting** - Técnica de engenharia social com criação de cenário falso
- **Protocolo** - Conjunto de regras para comunicação

## 🔹 Q
- **Quadro** - Frame, PDU da camada de enlace

## 🔹 R
- **RADIUS** - Remote Authentication Dial-In User Service, AAA (autenticação, autorização, contabilização)
- **Rainbow Table** - Tabela pré-computada de hashes
- **Ransomware** - Malware que sequestra dados e pede resgate
- **Recursão DNS** - Processo de consulta completa
- **Rede** - Conjunto de dispositivos interconectados
- **ReFS** - Resilient File System, foco em servidores
- **Repositório** - Local de armazenamento de pacotes (Linux)
- **RFC** - Request for Comments, documentos de padronização da internet
- **Risco** - Probabilidade de uma ameaça explorar uma vulnerabilidade
- **Rootkit** - Malware que esconde atividades no sistema
- **Roteador** - Equipamento camada 3, conecta redes diferentes
- **RSA** - Algoritmo de criptografia assimétrica (Rivest-Shamir-Adleman)

## 🔹 S
- **SaaS** - Software as a Service (software como serviço)
- **Salt** - Dado aleatório adicionado ao hash de senha
- **Scanner** - Ferramenta que varre redes/portas
- **Script Kiddie** - Hacker inexperiente que usa ferramentas prontas
- **Scrum** - Metodologia ágil com sprints
- **Scytale** - Cilindro usado para criptografia na Grécia antiga
- **Segmento** - PDU da camada de transporte
- **Senha** - Sequência de caracteres para autenticação
- **Servidor** - Computador que fornece serviços na rede
- **Setor** - Menor unidade do disco (512 bytes padrão)
- **SGBD** - Sistema Gerenciador de Banco de Dados
- **SHA** - Secure Hash Algorithm (SHA1, SHA256, SHA512)
- **SIEM** - Security Information and Event Management
- **SMB** - Server Message Block, compartilhamento de arquivos (porta 445)
- **SMTP** - Simple Mail Transfer Protocol (porta 25, envio de e-mail)
- **SNMP** - Simple Network Management Protocol, monitoramento
- **SOAR** - Security Orchestration, Automation and Response
- **Software** - Programas e sistemas
- **Spam** - Mensagem não solicitada em massa
- **Spear Phishing** - Phishing direcionado a alvos específicos
- **Spyware** - Malware que espiona atividades
- **SQL** - Structured Query Language, linguagem de banco relacional
- **SSH** - Secure Shell (porta 22, acesso remoto seguro)
- **SSL/TLS** - Secure Sockets Layer / Transport Layer Security, camada de segurança
- **Stuxnet** - Malware que atacou usinas nucleares iranianas
- **Sub-rede** - Divisão de uma rede IP
- **Super-rede** - Agregação de redes IP
- **Switch** - Equipamento camada 2, encaminha frames baseado em MAC
- **SYN** - Synchronize, pacote inicial do TCP
- **SYSLOG** - Protocolo de logs do sistema

## 🔹 T
- **TCP** - Transmission Control Protocol (orientado à conexão, confiável)
- **TCP/IP** - Conjunto de protocolos da internet
- **TDD** - Test-Driven Development, desenvolvimento orientado a testes
- **Telnet** - Acesso remoto inseguro (porta 23, texto claro)
- **Three-way Handshake** - SYN → SYN-ACK → ACK (estabelecimento TCP)
- **TLD** - Top Level Domain (ex: .com, .org)
- **Tolerância a falhas** - Capacidade de continuar operando mesmo com falhas
- **Tráfego** - Dados em transmissão na rede
- **Trojan** - Cavalo de troia, malware disfarçado
- **TTL** - Time to Live, tempo de vida do pacote

## 🔹 U
- **Ubuntu** - Distribuição Linux popular
- **UDP** - User Datagram Protocol (não orientado, não confiável)
- **UEFI** - Unified Extensible Firmware Interface, substituta da BIOS
- **Unicast** - Comunicação de um para um

## 🔹 V
- **Vírus** - Malware que se replica e precisa de hospedeiro
- **Vigenère** - Cifra polialfabética
- **Virtualização** - Simulação de hardware para rodar múltiplos sistemas
- **VLAN** - Virtual LAN, rede local virtual
- **VM** - Virtual Machine, máquina virtual
- **VPN** - Virtual Private Network, rede privada virtual
- **Vulnerabilidade** - Fraqueza que pode ser explorada

## 🔹 W
- **WAN** - Wide Area Network, rede de longa distância (ex: internet)
- **Whaling** - Phishing focado em alvos de alto escalão
- **White Hat** - Hacker ético autorizado
- **Wi-Fi** - Tecnologia wireless (IEEE 802.11)
- **Windows** - Sistema operacional da Microsoft
- **Wireshark** - Analisador de pacotes (sniffer)
- **Worm** - Malware auto-replicante (não precisa de hospedeiro)

## 🔹 X
- **XFS** - Sistema de arquivos de alta performance
- **XP** - Extreme Programming, metodologia ágil

## 🔹 Z
- **Zero Trust** - Modelo de segurança que não confia em nada por padrão
- **Zona de Disponibilidade** - Data centers isolados em uma região de nuvem
