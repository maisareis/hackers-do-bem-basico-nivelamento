# Aula 1 – Estrutura do IPv4

## 📌 Por que precisamos de IP
- Identificar dispositivos na rede
- Roteamento entre redes
- Comunicação fim-a-fim

## 📌 Estrutura
- 32 bits
- 4 octetos separados por pontos
- Cada octeto: 8 bits (0 a 255 em decimal)

## 📌 Exemplo
192.168.56.1
- 192 → primeiro octeto
- 168 → segundo octeto
- 56 → terceiro octeto
- 1 → quarto octeto

## 📌 Representação
- **Decimal:** 192.168.56.1
- **Binário:** 11000000.10101000.00111000.00000001
- **Hexadecimal:** C0.A8.38.01

## 📌 RFCs (Request for Comments)
- Documentos que padronizam a Internet
- RFC 791 → IPv4
- RFC 1918 → IPs privados
- RFC 2131 → DHCP
