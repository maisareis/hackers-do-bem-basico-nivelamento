# Aula 5 – Acesso Remoto

## 📌 O que é acesso remoto
- Acessar outro dispositivo à distância
- Mesma rede ou via internet
- Intensificado pela pandemia COVID-19

## 📌 Elementos necessários
- Software (cliente/servidor)
- Hardware (dispositivos)
- Conectividade de rede

## 📌 Tipos de acesso

### Remote Desktop (RDP)
- Área de trabalho remota
- Protocolo proprietário Microsoft
- Porta 3389
- Interface gráfica

### VPN (Virtual Private Network)
- Rede privada virtual
- Cliente se conecta à rede como se estivesse lá
- Túnel criptografado

## 📌 Protocolos

### Telnet
- Porta 23
- Acesso remoto por texto
- **Inseguro** (dados em texto claro)

### SSH (Secure Shell)
- Porta 22
- Criptografado (chaves assimétricas)
- Substitui Telnet
- Mais seguro

## 📌 Ferramentas populares
- AnyDesk
- TeamViewer
- LogMeIn Pro
- Microsoft Remote Desktop
- Real VNC
- KHelpDesk
