# Aula 3 – Protocolo TCP

## 📌 Características
- **Orientado à conexão** → estabelece conexão antes de enviar dados
- **Confiável** → confirma recebimento (ACK)
- **Reenvia dados perdidos**
- **Reorganiza segmentos fora de ordem**
- **Controle de fluxo** → evita sobrecarregar receptor

## 📌 Funções do TCP
- Segmentar dados para envio
- Remontar dados na ordem correta
- Confirmar recebimento
- Reenviar segmentos perdidos

## 📌 Quando usar TCP
- Web (HTTP/HTTPS)
- E-mail (SMTP, POP, IMAP)
- Transferência de arquivos (FTP)
- Acesso remoto (SSH)
- Qualquer aplicação que não pode perder dados

## 📌 Exemplos
- Navegar em sites
- Baixar arquivos
- Enviar e-mails
