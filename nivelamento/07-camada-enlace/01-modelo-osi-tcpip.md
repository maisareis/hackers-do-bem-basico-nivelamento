# Aula 1 – Modelo OSI e TCP/IP

## 📌 Processo de comunicação
- **Emissor** → quem envia
- **Receptor** → quem recebe
- **Mensagem** → o que é enviado
- **Canal** → meio de transmissão
- **Código/Protocolo** → regras da comunicação

## 📌 Modelo OSI (7 camadas)
1. **Física** → bits, cabos, sinais
2. **Enlace** → quadros, MAC, switches
3. **Rede** → pacotes, IP, roteadores
4. **Transporte** → segmentos, portas, TCP/UDP
5. **Sessão** → controle de diálogo
6. **Apresentação** → formatação, criptografia
7. **Aplicação** → dados do usuário (HTTP, DNS, SMTP)

## 📌 Modelo TCP/IP (4/5 camadas)
- **Aplicação** → (Aplicação + Apresentação + Sessão)
- **Transporte** → TCP, UDP
- **Internet** → IP, roteamento
- **Acesso à rede** → Enlace + Física

## 📌 PDU (Protocol Data Unit)
- **Dados** → camada de Aplicação
- **Segmento** → camada de Transporte
- **Pacote** → camada de Rede
- **Quadro (Frame)** → camada de Enlace
- **Bits** → camada Física

## 📌 Organizações padronizadoras
- **IEEE** → padrões Ethernet (802.3)
- **IETF** → RFCs da Internet
- **ISO** → modelo OSI
