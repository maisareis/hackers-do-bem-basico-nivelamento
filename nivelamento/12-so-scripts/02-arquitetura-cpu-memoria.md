# Aula 2 – Arquitetura de Hardware: CPU, Memória, BIOS

## 📌 CPU (Processador)
- Central Processing Unit
- Processa dados e transforma em informação

### Características
- **Frequência (clock)** → velocidade (GHz)
- **Núcleos** → mais núcleos = mais tarefas simultâneas
- **Cache** → memória rápida entre CPU e RAM (L1, L2, L3)

### Arquiteturas
- **x86 (32 bits)** → CISC
- **x64 (64 bits)** → CISC
- **ARM** → RISC (móveis, IoT)

### Fabricantes
- **Intel:** Core i3, i5, i7, i9 (1ª a 13ª geração)
- **AMD:** Ryzen 3, 5, 7, 9 (séries 1000 a 7000)
- **Apple:** M1, M2, M3

## 📌 Placa-mãe
- Conecta todos os componentes
- **Socket** → encaixe do processador
- **Chipset** → controla comunicação (Intel H/B/Z, AMD A/B/X)
- **Slots** → PCIe, memória RAM, M.2
- **Conectores** → SATA, USB, áudio, rede

## 📌 Memória RAM
- Random Access Memory
- Volátil (perde dados ao desligar)
- Tipos: DDR, DDR2, DDR3, DDR4, DDR5
- Capacidade: GB
- Frequência: MHz

### Memórias de vídeo
- GDDR3, GDDR5, GDDR6, GDDR6X
- HBM, HBM2

## 📌 BIOS/UEFI
- Basic Input Output System
- Inicializa hardware ao ligar
- **POST** (Power-On Self-Test)
- Configurações: boot, virtualização, overclock
- UEFI substitui BIOS (interface gráfica, mais recursos)

### Teclas comuns para acessar
- **ASUS:** F2 ou DEL
- **Dell:** F2 ou F12
- **HP:** F10
- **Lenovo:** F1, F2, Fn+F2
