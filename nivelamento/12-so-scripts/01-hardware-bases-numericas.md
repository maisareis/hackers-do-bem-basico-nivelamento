# Aula 1 – Hardware e Bases Numéricas

## 📌 Grandezas computacionais
- **Bit (b)** → Binary Digit, 0 ou 1
- **Byte (B)** → 8 bits
- **KB (Kilobyte)** → 1000 bytes (SI) ou 1024 bytes (IEC)
- **MB (Megabyte)** → 1000 KB ou 1024 KB

## 📌 Sistemas numéricos

### Base 2 (Binário)
- Dígitos: 0, 1
- Usado internamente pelo computador

### Base 10 (Decimal)
- Dígitos: 0 a 9
- Usado por humanos

### Base 16 (Hexadecimal)
- Dígitos: 0-9, A(10), B(11), C(12), D(13), E(14), F(15)
- Usado para representar endereços MAC, IPv6, cores

## 📌 Conversões

### Binário → Decimal
1011₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8 + 0 + 2 + 1 = 11₁₀

### Hexadecimal → Binário
FACA₁₆ = F(1111) A(1010) C(1100) A(1010) = 1111101011001010₂

## 📌 Grandezas de armazenamento
- **Bit (b)** → menor unidade
- **Byte (B)** → 8 bits
- **KB** → 1024 bytes
- **MB** → 1024 KB
- **GB** → 1024 MB
- **TB** → 1024 GB

## 📌 Grandezas de transmissão
- **Kbps** → kilobits por segundo
- **Mbps** → megabits por segundo
- **KB/s** → kilobytes por segundo
- **MB/s** → megabytes por segundo

## 📌 Grandezas elétricas
- **Hz** → frequência (clock)
- **V** → tensão
- **A** → corrente
- **W** → potência
