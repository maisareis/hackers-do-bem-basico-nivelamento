# Aula 11 – Linux: Versões e Distribuições

## 📌 História
- 1991: Linus Torvalds cria o Linux (baseado no MINIX)
- Kernel do sistema operacional
- Open source

## 📌 Distribuições Linux

### Slackware (1993)
- Mais antiga e tradicional
- Instalação e gerenciamento manuais
- Controle total ao administrador

### Debian
- Base para muitas outras distros
- Gerenciador: dpkg/apt
- Estável, seguro, personalizável

### Ubuntu (base Debian)
- Mais utilizada
- Fácil, grande comunidade
- Compatibilidade de hardware

### Linux Mint (base Ubuntu)
- Interface lembra Windows
- Migração mais fácil

### Kali Linux (base Debian)
- Focada em pentest e forense
- Ferramentas pré-instaladas

### ParrotOS (base Debian)
- Pentest e privacidade

### Arch Linux
- Pacotes: pacman
- Rápido, estável, seguro
- Instalação não amigável

### Manjaro (base Arch)
- Arch para iniciantes
- Crescendo em popularidade

### Fedora (base Red Hat)
- Patrocinado pela Red Hat
- Atualizações rápidas
- Gerenciador: dnf (RPM)

### Android
- Kernel Linux
- Base para dispositivos móveis
- ROMs modificadas

## 📌 Desktop x Servidor
- **Servidor:** modo texto, sem interface gráfica, eficiente
- **Desktop:** interface gráfica, facilidade para usuários
