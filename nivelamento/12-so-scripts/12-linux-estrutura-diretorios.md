# Aula 12 – Linux: Estrutura de Diretórios

## 📌 / (raiz)
- Diretório principal
- Todos os outros ficam dentro

## 📌 /bin
- Binários essenciais (comandos básicos)
- ls, cp, mv, cat, etc.

## 📌 /sbin
- Binários de sistema (administração)
- ip, route, fdisk, useradd

## 📌 /boot
- Kernel e arquivos de inicialização
- GRUB, initrd

## 📌 /dev
- Dispositivos (arquivos especiais)
- sda (disco), tty (terminais), random

## 📌 /etc
- Configurações do sistema
- passwd, shadow, hosts, network

## 📌 /home
- Diretórios dos usuários
- /home/usuario (Desktop, Downloads, Documentos)

## 📌 /root
- Diretório do usuário root

## 📌 /usr
- Programas e bibliotecas do usuário
- /usr/bin, /usr/lib, /usr/share

## 📌 /var
- Dados variáveis
- Logs (/var/log), spool, cache

## 📌 /tmp
- Arquivos temporários
- Apagados na reinicialização

## 📌 /media
- Montagem automática de dispositivos (pendrives, CDs)

## 📌 /mnt
- Montagem manual de sistemas de arquivos

## 📌 /proc
- Informações de processos e kernel (virtual)
- CPU, memória, processos em execução

## 📌 /sys
- Informações de dispositivos (similar ao /proc)
- Hardware, drivers

## 📌 /run
- Dados de programas em execução
- PIDs, sockets
