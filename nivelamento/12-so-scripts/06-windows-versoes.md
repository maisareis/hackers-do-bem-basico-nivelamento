# Aula 6 – Windows: Versões

## 📌 Histórico do Windows

### Windows 3.11 (1992)
- Melhorias em multimídia
- Gerenciador de arquivos (Windows Explorer)
- Ctrl+C / Ctrl+V

### Windows 95 (1995)
- Nova interface gráfica
- Botão Iniciar!
- Plug-and-Play
- Internet Explorer
- 32 bits

### Windows 98 (1998)
- Otimizado para internet
- APIPA, DHCP
- Scandisk, Desfragmentador, SFC

### Windows ME (2000)
- Movie Maker
- Nova arquitetura TCP/IP
- Windows Restore, Update
- Muitos bugs

### Windows XP (2001)
- Interface mais moderna (ClearType)
- Verificação de autenticidade
- Wi-Fi, DSL, IPv6
- 32 e 64 bits

### Windows Vista (2006)
- Interface Aero (transparências)
- Windows Defender
- Sistema pesado, muitos popups

### Windows 7 (2009)
- Aero melhorada
- Otimizado para múltiplos núcleos
- Inicialização mais rápida
- Virtual PC

### Windows 8 (2012)
- Interface Metro (tablets)
- Compatível com UEFI e Secure Boot
- USB 3.0, NVMe
- Remoção do menu iniciar (voltou no 8.1)

### Windows 10 (2015)
- Interface híbrida
- Atualizações contínuas (como serviço)
- Windows Subsystem for Linux (WSL)

### Windows 11 (2021)
- Interface redesenhada
- Barra de tarefas centralizada
- TPM 2.0 obrigatório
- Aplicativos Android
- 64 bits apenas
