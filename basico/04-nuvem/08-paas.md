# Aula 8 – PaaS (Platform as a Service)

## 📌 O que é PaaS
- Platform as a Service
- Plataforma para desenvolvimento, execução e gerenciamento de aplicativos
- Não precisa se preocupar com infraestrutura subjacente

## 📌 Ferramentas oferecidas
- Ambientes de desenvolvimento integrados (IDEs)
- Bancos de dados
- Serviços web

## 📌 Comparativo: criar banco de dados
**Em IaaS:**
- Provisionar servidor (VM)
- Instalar e configurar banco de dados
- Gerenciar atualizações

**Em PaaS:**
- Plataforma cria ambiente automaticamente
- Desenvolvedor só adiciona os dados

## 📌 Responsabilidades no PaaS
| Responsabilidade | Quem faz |
|-----------------|----------|
| Informações e dados | Cliente |
| Dispositivos (móveis/PCs) | Cliente |
| Contas e identidades | Cliente |
| Aplicativos | Cliente |
| Demais camadas | Provedor |

## 📌 Vantagens
- **Agilidade** → foco no desenvolvimento
- **Escalabilidade** → fácil dimensionamento
- **Colaboração** → equipes trabalham juntas
