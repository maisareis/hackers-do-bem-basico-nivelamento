# Aula 7 – IaaS (Infrastructure as a Service)

## 📌 O que é Infraestrutura
- Base tecnológica para suportar sistemas e redes
- Inclui servidores, redes, armazenamento
- Fornece capacidade computacional e conectividade

## 📌 O que é IaaS
- Infrastructure as a Service
- Recursos de infraestrutura virtualizados pela internet
- Clientes alugam servidores, redes, armazenamento
- Pagamento por uso, sem adquirir hardware físico

## 📌 Responsabilidades no IaaS
| Responsabilidade | Quem faz |
|-----------------|----------|
| Informações e dados | Cliente |
| Dispositivos (móveis/PCs) | Cliente |
| Contas e identidades | Cliente |
| Aplicativos | Cliente |
| Controles de rede | Cliente |
| Sistema Operacional | Cliente |
| Hosts físicos | Provedor |
| Rede física | Provedor |
| Datacenter | Provedor |

## 📌 Exemplo clássico
- Locação de máquinas virtuais
- Criar e gerenciar VMs para hospedar aplicativos
- Aumentar/diminuir recursos conforme demanda
- Acessar de qualquer lugar via área de trabalho remota
