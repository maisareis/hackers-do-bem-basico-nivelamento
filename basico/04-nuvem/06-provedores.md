# Aula 6 – Principais Provedores de Nuvem

## 📌 Amazon Web Services (AWS)
- Fundada em 2006
- Líder global, mais de 200 serviços
- Infraestrutura global com múltiplas regiões
- **Serviços principais:** EC2, S3, Elastic Beanstalk, Lambda, RDS

## 📌 Microsoft Azure
- Lançado em 2010
- Forte integração com ambiente Microsoft
- Globalmente distribuído
- **Serviços principais:** Virtual Machines, Blob Storage, SQL Database, Functions, App Service

## 📌 Google Cloud Platform (GCP)
- Lançado em 2008
- Inovação tecnológica, interface intuitiva
- Preços competitivos
- **Serviços principais:** Compute Engine, Cloud Storage, Cloud Functions, AI/ML, App Engine

## 📌 Comparação
- **AWS:** mais estabelecido, maior variedade
- **Azure:** melhor para empresas que usam tecnologias Microsoft
- **GCP:** inovação, preços competitivos
