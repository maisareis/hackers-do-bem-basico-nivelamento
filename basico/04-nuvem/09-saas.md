# Aula 9 – SaaS (Software as a Service)

## 📌 O que é SaaS
- Software as a Service
- Acesso a aplicativos via internet (assinatura)
- Não precisa comprar, instalar ou manter localmente
- Usa direto pelo navegador

## 📌 Comparativo: armazenar arquivo
**Em PaaS:**
- Escolher plataforma
- Configurar ambiente
- Implementar serviço
- Criar aplicativo

**Em SaaS:**
- Criar conta
- Fazer login
- Começar a usar

## 📌 Responsabilidades no SaaS
| Responsabilidade | Quem faz |
|-----------------|----------|
| Informações e dados | Cliente |
| Dispositivos (móveis/PCs) | Cliente |
| Contas e identidades | Cliente |
| Demais camadas | Provedor |

## 📌 Vantagens
- Mobilidade
- Atualizações automáticas
- Economia de custos
- Foco nos objetivos
- Sem complexidade técnica
