# Aula 13 – Serviços de Banco de Dados

## 📌 Problemas comuns
- Dificuldade em gerenciar estoque
- Dificuldade em gerenciar pedidos e dados de clientes
- Dificuldade em gerar recomendações

## 📌 Solução: Banco de Dados
- Registro organizado de produtos
- Gestão de disponibilidade
- Controle de clientes
- Histórico de compras

## 📌 Banco de Dados Local
- Servidores físicos necessários
- Custo inicial maior
- Escalabilidade limitada
- Acesso somente local

## 📌 Banco de Dados na Nuvem
- Pagamento sob demanda
- Custo inicial menor
- Escalabilidade ilimitada
- Acesso remoto de qualquer lugar

## 📌 Serviços de Banco de Dados na Nuvem
- **AWS:** RDS (Relational Database Service)
- **Azure:** SQL Database
- **GCP:** Cloud SQL
