# Aula 12 – Serviços de Computação

## 📌 Máquinas Virtuais (VMs)
- Ambientes virtuais que emulam hardware e SO
- Executam aplicativos como se fossem físicos
- Várias VMs em um único servidor físico
- Cada VM tem próprio SO, CPU, memória, armazenamento

## 📌 Provedores de VM
- **AWS:** EC2 (Elastic Compute Cloud)
- **Azure:** Virtual Machines
- **GCP:** Compute Engine

## 📌 Serverless (Computação sem servidor)
- Executar aplicativos sem gerenciar servidores
- Provedor gerencia infraestrutura automaticamente
- Foco na codificação
- Funções ativadas por eventos

## 📌 Provedores Serverless
- **AWS:** Lambda
- **Azure:** Functions
- **GCP:** Cloud Functions

## 📌 Vantagens
- Criar servidores a qualquer momento
- Acessar de qualquer lugar
- Serverless: foco no serviço, sem preocupação com infraestrutura
