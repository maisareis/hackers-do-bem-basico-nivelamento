# Aula 14 – Formas de interagir com a nuvem (CLI, SDK, Console)

## 📌 Console Web
- Interface gráfica baseada em navegador
- Configuração, monitoramento e gerenciamento visual
- **Quando usar:**
  - Ações pontuais rápidas
  - Configuração visual
  - Usuários que preferem GUI

## 📌 CLI (Command Line Interface)
- Interface de linha de comando
- **Quando usar:**
  - Automação de tarefas repetitivas
  - Configuração inicial rápida
  - Desenvolvimento (ambiente de codificação)
  - Scripting avançado

## 📌 SDK (Software Development Kit)
- Conjunto de ferramentas, bibliotecas e recursos
- Facilita desenvolvimento de aplicativos que interagem com a nuvem
- Oferece funções, classes e métodos predefinidos
- Permite usar linguagens de programação familiares

## 📌 Resumo
- **Console Web:** visual e intuitivo
- **CLI:** eficiência e automação
- **SDK:** desenvolvimento simplificado
