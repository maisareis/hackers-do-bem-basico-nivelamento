# Aula 2 – Histórico

## 📌 Década de 1960
- Conceito de "time-sharing" (compartilhamento de tempo)
- Várias pessoas compartilhavam um único computador central

## 📌 Década de 1990
- Popularização da Internet
- Conexão global de dispositivos

## 📌 Virtualização
- Criação de múltiplas máquinas virtuais em um servidor físico
- Maior eficiência no uso de recursos

## 📌 Anos 2000
- Surgem os primeiros provedores de nuvem
- **2006:** Amazon lança AWS (aproveitando recursos subutilizados)
- Google e Microsoft entram no mercado

## 📌 Como surgiu a AWS
- Amazon construiu infraestrutura robusta para seu e-commerce
- Recursos ficavam subutilizados
- Ideia: alugar recursos excedentes para outras empresas
