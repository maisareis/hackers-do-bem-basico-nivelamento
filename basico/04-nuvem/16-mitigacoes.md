# Aula 16 – Segurança em Nuvem: Mitigando falhas

## 📌 Mitigando Acesso Não Autorizado
- **IAM** (Identity and Access Management)
- Políticas, processos e tecnologias para gerenciar identidades e permissões
- Apenas usuários autorizados acessam recursos

### Serviços de IAM
- **AWS:** IAM
- **Azure:** Active Directory (Azure AD)
- **GCP:** Cloud IAM

## 📌 Mitigando Configurações Inadequadas
- **Configuração Segura**
- Avaliação e ajustes constantes
- Conformidade com padrões de segurança
- Evitar vulnerabilidades

### Serviços de Configuração
- **AWS:** AWS Config
- **Azure:** Azure Policy
- **GCP:** Security Command Center

## 📌 Mitigando Falta de Monitoramento
- **Sistema de Monitoramento**
- Observar desempenho, status e atividades em tempo real
- Identificar problemas e atividades suspeitas

### Serviços de Monitoramento
- **AWS:** CloudWatch
- **Azure:** Azure Monitor
- **GCP:** Cloud Operations Suite
