# Aula 11 – Serviços de Armazenamento

## 📌 Armazenamento tradicional
- Dispositivos físicos (pen drives, HDs, cartões de memória)
- Dependente do dispositivo

## 📌 Armazenamento em Nuvem
- Guardar e acessar arquivos de qualquer lugar
- Sem depender de dispositivo físico

## 📌 Tipos de Armazenamento

### Armazenamento de Objetos
- Grandes quantidades de dados (fotos, vídeos, backups)
- Exemplo: Amazon S3
- Estrutura flexível, qualquer "objeto"

### Armazenamento de Blocos
- Discos virtuais de alto desempenho
- Vinculados a máquinas virtuais
- Alta I/O
- Exemplo: Azure Disk Storage

## 📌 Classes de Armazenamento (AWS)
- **S3 Standard:** alta disponibilidade, ≥3 zonas
- **S3 Intelligent-Tiering:** otimização automática de custos
- **S3 Standard-IA:** acesso infrequente
- **S3 One Zone-IA:** dados recriáveis, zona única

## 📌 Vantagens
- Escalabilidade
- Redundância
- Acessibilidade
- Gerenciamento simplificado
