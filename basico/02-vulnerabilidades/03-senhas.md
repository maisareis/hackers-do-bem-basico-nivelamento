# Aula 3 – Vulnerabilidades em Senhas

## 📌 Importância das senhas
- Controle de acesso
- Proteção de dados pessoais
- Prevenção contra fraudes e roubo de identidade
- Segurança empresarial
- Acesso a dispositivos eletrônicos

## 📌 Tipos de vulnerabilidades em senhas
- **Senhas fracas** → curtas, óbvias, baseadas em informações pessoais
- **Reutilização** → mesma senha em múltiplos serviços
- **Armazenamento inseguro** → senhas em texto claro
- **Ataques de força bruta** → testar combinações até acertar
- **Engenharia social** → enganar usuário para revelar a senha

## 📌 Boas práticas
- Complexidade (maiúsculas, minúsculas, números, símbolos)
- Comprimento adequado (mínimo 8 caracteres)
- Evitar informações pessoais (nome, data de nascimento)
- Não usar palavras comuns do dicionário
- Não reutilizar senhas entre serviços
- Usar gerenciador de senhas
- Ativar autenticação de dois fatores (2FA/MFA)
