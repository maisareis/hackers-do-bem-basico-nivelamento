# Aula 14 – Chave Pública e Privada (parte 2)

## 📌 Algoritmos de criptografia assimétrica
- **DIFFIE-HELLMAN:** troca de chaves simétricas de forma segura
- **RSA:** criptografia de dados
- **ECC:** criptografia de dados (curvas elípticas)
- **DSA:** assinaturas digitais

## 📌 Vantagens
- Não há problema de compartilhamento de chaves
- Mais segura que criptografia simétrica

## 📌 Desvantagens
- Algoritmos pesados
- Lentidão na comunicação
- Consomem muitos recursos da máquina

## 📌 Por que não usar só assimétrica?
- Simétrica tem seu espaço por ser mais leve
- Algoritmos mais simples e rápidos
- Cada tipo adequado para determinada situação

## 📌 Conclusão
- Não existe "melhor" sistema
- Características diferentes para situações diferentes
- Cabe ao profissional escolher o adequado
