# Aula 10 – Protocolos Seguros

## 📌 Visualização com Wireshark
- Capturar tráfego de rede
- Analisar protocolos

## 📌 HTTP (sem criptografia)
- Dados visíveis em texto claro
- Possível ler requisições e respostas
- Informações sensíveis expostas

## 📌 HTTPS (com criptografia)
- Dados visíveis mas incompreensíveis
- Payload criptografado
- Não é possível extrair informações

## 📌 SSL/TLS
- Protocolo responsável pela camada extra de segurança
- Transforma HTTP em HTTPS
- Usado também em outros protocolos (FTP, SMTP)

## 📌 Importância
- Dominar SSL/TLS é essencial para profissional de segurança
- Base para entender criptografia em comunicações
