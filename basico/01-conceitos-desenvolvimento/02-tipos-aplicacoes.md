# Aula 2 – Principais tipos de aplicações

## 📌 Compiladores e Interpretadores
- **Compiladas** → transformadas diretamente em código de máquina (C, C++, Java, C#)
  - Mais rápidas, mais controle sobre hardware
- **Interpretadas** → executam instrução por instrução (Python, JavaScript, Ruby, PHP)

## 📌 O que é uma aplicação
- Converte tarefas em processos digitais, simplificando atividades cotidianas

## 📌 Tipos de aplicações
- **Aplicações web** → acessadas via navegador (JavaScript, HTML, CSS)
- **Aplicações mobile** → dispositivos móveis (Swift iOS, Kotlin/Java Android, React Native)
- **Aplicações desktop** → instaladas localmente (Java, C#, Python)
- **Aplicações híbridas** → combinam web e nativo, multiplataforma (React Native, Ionic)
- **Aplicações nativas** → específicas de uma plataforma (Swift, Kotlin, C# UWP)
- **Aplicações em nuvem** → hospedadas em servidores remotos (Java, Python, Node.js)
- **Aplicações IA** → algoritmos inteligentes (Python, R)
- **Aplicações RV/RA** → realidade virtual e aumentada (Unity C#, Java, Swift)
