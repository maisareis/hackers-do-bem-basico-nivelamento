# Aula 1 – Tipos de linguagem de programação

## 📌 Linguagem de Programação
- Ferramenta que segue instruções para construir programas
- Comunicação entre programador e computador
- Especifica: dados de entrada, tratamento, armazenamento, ações

## 📌 Linguagem de programação x Linguagem de computação
- **Linguagem de programação** → conjunto de regras interpretadas pelo programador
- **Linguagem de computação** → inclui linguagens que formatam texto (HTML, CSS)
- Nem toda linguagem computacional é de programação

## 📌 Tipos de linguagem
- **Baixo nível** → comunicação direta com a máquina, exige conhecimento de hardware
  - **Linguagem de máquina** → binário (0 e 1)
  - **Assembly** → instruções como MOV, ADD
- **Alto nível** → instruções mais próximas da linguagem humana (Java, Python, JavaScript)
  - Usam tradutores ou compiladores
  - Vantagens: redução do tempo, entendimento mais fácil, não precisa atuar na operação interna

## 📌 Tradutor vs Compilador
- **Tradutor** → traduz e executa à medida que é traduzido
- **Compilador** → traduz programa inteiro de uma vez, pode ser armazenado

## 📌 Linguagens mais utilizadas
- **Java** → portável, escalável, base do Android
- **JavaScript** → front-end e back-end, versátil
- **C#** → portável, escalável, desenvolvimento de jogos
- **Python** → fácil, sintaxe simples, Machine Learning, Ciência de Dados
- **Ruby** → sintaxe simples, usado em Airbnb, Twitter, GitHub
